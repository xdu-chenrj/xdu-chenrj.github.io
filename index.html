二叉树中的最大路径和

发表于 2023-08-28 分类于 leetcode

题意

二叉树中的 路径 被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中 至多出现一次 。该路径 至少包含一个 节点，且不一定经过根节点。

路径和 是路径中各节点值的总和。

给你一个二叉树的根节点 root ，返回其 最大路径和 。

题解

定义一个dfs函数，表示从当前节点开始，往下搜的最大值;

对于当前节点node来说，dfs(node->left) + dfs(node->right) + node->value就是以当前节点为中枢的最大路径和;

对于当前节点node来说，左儿子的最大路径和left = dfs(node->left)， 右儿子的最大路径和right = dfs(node->right)，因为左儿子和右儿子的最大路径和有负数的情况，也就是说节点的值全部是负数，这个时候就看情况取舍左右儿子的最大路径和了，或者直接left = max(0, left); right = max(0, right);;

定义一个变量维护最终答案， 对每个节点求出左儿子的最大路径和右儿子的最大路径和，然后更新最大值。ans = max(ans, left + right + node->val)，因为有可能全是负节点，那么最大路径和就是负值最小的节点，所以ans的值要定义成一个较小的负数，比如-1e9；
